# Kaiso.ai

**AI-powered data transformation studio for analysts who need transparent, auditable, and reproducible data pipelines.**

Describe your transformations in plain language. Kaiso generates Python code you can read, verify, and share — step by step. Your data never leaves your browser; only column names are sent to the AI.

## How It Works

1. **Connect** — Drop in CSV, Excel, or JSON files
2. **Transform** — Describe changes in plain language; Kaiso generates Python
3. **Verify** — Inspect each step with automatic issue detection
4. **Share** — Export the pipeline, not the data, for full reproducibility

## Principles

- **Transparent** — Every transformation is readable Python, not a hidden prompt
- **Private** — Data stays in your browser; only metadata reaches the AI
- **Reproducible** — Same input, same output, every time
- **Auditable** — Built-in methodology reporting and verification testing
